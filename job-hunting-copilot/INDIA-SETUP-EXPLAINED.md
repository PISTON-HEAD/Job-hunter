## 🎯 WHAT THE AI WON'T DO

The cover letter generator follows strict rules:

### ❌ Won't Lie About Your Skills:

- **Job requires:** Python, Django, React
- **You have:** Java, Spring Boot, Docker
- **AI will NOT say:** "I have extensive Python and React experience"
- **AI will ONLY mention:** Java, Spring Boot, Docker (your actual skills)

### ❌ Won't Use Fake Projects:

- Shows only what's in your GitHub (github.com/piston-head)
- Won't invent achievements you didn't tell it about
- Won't claim certifications you don't have

### ❌ Won't Oversell:

- **Job says:** "10+ years experience required"
- **You have:** 2-4 years
- **AI won't say:** "I'm a senior expert with decades of experience"
- **AI will filter this job out** (score below 75) so you don't waste time

## ❓ COMMON QUESTIONS

### Q: Will this get me banned from LinkedIn/Naukri?

**A:** No! You're using public APIs and manually applying. No automation of clicks or account actions.

### Q: Do I need to keep my computer on all day?

**A:** Only if you're self-hosting N8N. Options:

- Run on a cheap cloud server (AWS/DigitalOcean for ₹400/month)
- Use N8N cloud (₹800/month)
- Or just run it manually whenever you want (don't need daily schedule)

### Q: Can I customize the cover letters before applying?

**A:** YES! That's recommended. The AI gives you a starting point, you should always:

- Read the cover letter
- Add personal touches
- Fix anything that doesn't sound like you
- Remove parts that don't fit

### Q: What if I don't have GitHub projects?

**A:** Remove the GitHub link from the cover letter template. Focus on work experience instead.

### Q: Can I search for different jobs (not Java)?

**A:** Yes! Edit the search keywords in Nodes 2-6:

- Change "Java Spring Boot" → "Python Django"
- Change "Java Spring Boot" → "React Frontend"
- Change "Java Spring Boot" → "DevOps Engineer"

### Q: Will cover letters look AI-generated?

**A:** Not with the new simplified prompt! They're short, simple, and natural. But you should still:

- Read each one
- Add your personality
- Make small edits so it sounds like YOU

### Q: How many jobs should I apply to per day?

**A:** Quality > Quantity. The workflow finds 10-15 high-match jobs. Apply to 3-5 per day with customized applications rather than 50 generic ones.
